# Priority Rules for Conflict and Tragic Trade-offs

## Purpose

This file defines how Invest-GT should handle cases where priorities collide and no clean option exists.

Some decisions are not optimization problems but tragedy-management problems. In those cases, the repository should make the damage and the reason for choosing a path visible.

---

## Core claim

When all available actions involve some harm, the framework should prefer:
1. the lower total avoidable damage,
2. the lower irreversible damage,
3. the lower concentration of harm on identifiable persons,
4. the option that preserves future repair capacity,
5. the option that avoids turning necessity into ideology.

---

## Main cases

### Case A: Self vs. others

A person may face an action that improves personal stability while imposing real cost on close others.

The record should state:
- why the personal need is real,
- what cost falls on others,
- whether alternatives exist,
- what mitigation or repair is owed.

### Case B: One subgroup vs. another

A group may stabilize one subgroup by shifting cost onto another.

The framework should ask whether this is:
- a temporary emergency measure,
- a justified burden under necessity,
- or a disguised extraction pattern.

### Case C: Lesser evil under pressure

In some settings the right question is not how to avoid all harm, but which path produces less destruction.

This should be recorded as tragic choice, not celebrated as pure success.

---

## On “Kantisierung” as risk

The repository should explicitly watch for a failure mode in which rigid moral framing hides real-world consequences.

Here the term can be used as a warning label for situations where:
- formal purity is preserved,
- but practical damage rises,
- and no responsible lesser-evil analysis is performed.

This warning is not a rejection of moral principle. It is a reminder that a framework must still account for concrete consequences, damaged persons, and repair obligations.

---

## Review questions

When priorities conflict, ask:
- Which harms are unavoidable?
- Which harms are still avoidable?
- Which option preserves the most repair capacity?
- Which option most unfairly concentrates burden?
- Which option depends on hiding victims from the record?
- Has a rigid rule replaced actual damage analysis?
