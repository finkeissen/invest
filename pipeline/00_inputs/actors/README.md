# Actors

This folder defines the entities that matter strategically inside the modeled world.

Use this folder for actor taxonomies and actor definitions across multiple scales.

## Main actor levels

- persons
- households and families
- communities and groups
- institutions and firms
- states and alliances
- non-state disruptive actors

## Each actor description should clarify

- role
- incentives
- constraints
- resources
- time horizon
- relevance to later agent modeling

## Design note

A group is not just a large person.

A person, a household, a company, a church, and a state all invest differently. They face different constraints, different forms of gain, and different kinds of failure.
