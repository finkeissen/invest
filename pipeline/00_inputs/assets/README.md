# Assets

This folder defines the asset universe that links investment reasoning to practical allocation.

Use this folder for financial assets, productive assets, human capital assets, and collective capital assets.

## Core idea

An asset is anything that can preserve, generate, amplify, or protect future gain.

Not all assets are market-traded.

## Typical asset families

- financial assets
- productive assets
- health assets
- educational assets
- time and attention assets
- relationship and trust assets
- institutional assets
- strategic and geopolitical assets
